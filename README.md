# Task Management System

A Spring Boot-based task management system with user and admin workflows.

## Features
- User registration and login
- Task creation, editing, and viewing
- Admin dashboard and task management
- Password reset workflow

## Technology Stack
- Java 21
- Spring Boot 3.4.7
- Spring Security
- Spring Data JPA
- JSP views
- Maven

## Run locally
```bash
./mvnw spring-boot:run
```
